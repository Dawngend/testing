# Local Installation and Setup Guide

Follow these step-by-step instructions to clone the subfolder from the repository and get the pure Python web application running locally.

## Prerequisites

Before you begin, ensure you have the following installed on your machine:
* **Git** (For cloning the repository)
* **Python** (v3.10 or higher recommended)
* **pip** (Python package installer)

## Step-by-Step Installation

### 1. Clone the Repository
Because this project lives inside a subdirectory of a personal projects repository, clone the main branch and navigate into the project folder:

```bash
# Clone the repository
git clone [https://github.com/Dawngend/Personal-Projects.git](https://github.com/Dawngend/Personal-Projects.git)

# Navigate directly into the All In One Reviewer project directory
cd "Personal-Projects/All In One Reviewer"
